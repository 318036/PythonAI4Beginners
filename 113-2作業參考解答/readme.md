## 📚 作業一：加權總成績計算與成績等第評定

- 📝 題目連結：[作業一 - 加權總成績與成績等第](https://judgecode.org/problem/0/11111)
- 🎥 解說影片：[點此觀看](https://youtu.be/kiCeVAwfDf0)

> **說明**  
> 輸入五個科目的成績與對應的權重，計算加權總成績（小數點後兩位），並根據成績評定等第（如 A、B+、C 等）。  
> 需要處理多種輸入錯誤狀況，包括：
> - 成績數量不正確
> - 成績範圍錯誤（應在 0 ~ 100 之間）
> - 非有效數字的輸入  
> 👉 **學習目標**：練習多重輸入處理、浮點數運算、條件判斷與例外處理 (`try-except`)。
